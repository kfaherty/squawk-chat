# squawk-chat

## Prerequisites

You will need the following things properly installed on your computer:

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/)

## Installation

You will need to set up and install the project dependencies:

* `cd squawk-chat`
* `yarn install`

## Running / Development

* `cd squawk-chat`
* `yarn run start`
